A* algorithm exercises

Cho ma trận 2D đại diện cho một hình ảnh với các giá trị màu khác nhau (0–9). Áp dụng **thuật toán Flood Fill** để tô lại vùng liên thông bắt đầu từ điểm (x = 1, y = 1) có cùng màu với ô ban đầu bằng màu mới `9`. 

                Input:
                    image = [
                    [1, 1, 1, 2, 2],
                    [1, 1, 0, 2, 2],
                    [1, 0, 0, 2, 2],
                    [2, 2, 2, 2, 2]
                    ]
                    start = (1, 1)
                    newColor = 9

Kết quả:

  • Bắt đầu từ điểm (x, y).
  • Nếu ô hiện tại có cùng màu với màu gốc ban đầu → đổi thành màu mới.
  • Tiếp tục gọi đệ quy hoặc duyệt BFS/DFS cho 4 ô lân cận (trên, dưới, trái, phải).
  • Dừng khi không còn ô nào cùng màu ban đầu nữa.
    •                         Output:
                        [
                        [9, 9, 9, 2, 2],
                        [9, 9, 0, 2, 2],
                        [9, 0, 0, 2, 2],
                        [2, 2, 2, 2, 2]
                        ]
